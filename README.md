# MindTrack-Beta1
Field
Your Answer
App Name
MindTrack


App Type
Calendar and habit tracker


Problem It Solves
MindTrack helps you tackle everyday obstacles one step at a time. Through schedule management and proper planning you can take control of every second of the day. Reinforcing that idea is a built in habit tracker that works to keep you productively constructing new paths to success. 


Page 1 — Name & Description
Todays Dashboard (landing page): This page serves as the first page the user interacts with, it includes a time table with the events planned for the day. the habit checklist along with a visual showing the habit completion percentage for the day, and a day focus that shows screen time for the day.


Page 2 — Name & Description
Weekly view: This page shows the events planned for the week with a time table.


Page 3 — Name & Description
Monthly view: This page shows the events planned for the month with a month at a glance tab that has a drop down menu for 1m, 6m, and 12m which tracks the habit completion rates across the selected timespan.


Page 4 — Name & Description (optional but recommended)
Add event: This page is accessible for all other pages and functions to create new events at a selected time.


Key Interactive Feature 1
Weekly view/monthly view: highlighted when hovered over and takes the user to the selected page.


Key Interactive Feature 2
Previous month/week/day , Next month/week/day: highlighted when hovered over and takes the user to the selected timeframe. 


Key Interactive Feature 3
Add event (+): A button that when pressed is highlighted and takes the user to the add event page. 

Event Marks (.): Colored circles above the day of an event on monthly view.
